Detailed Description

The main CLIENT SIDE class
The purpose of this class is to create the client and the WEBTARGET as well as to initialize the first window
All the methods defined in this class are a bridge to the server side where the real actions are performed, none of this methods acces the DB they call the server side methods that acces them.
This class has a resourceBoundle that detects the lenguage of the computer and stores the information inside the logger accordly
